O que é computação em nuvem?

Computação em nuvem é a entrega de recursos de TI pela internet — como servidores, armazenamento, bancos de dados, redes e softwares — de forma sob demanda. Em vez de comprar e manter infraestrutura própria, você utiliza serviços em data centers remotos, pagando conforme o uso e ganhando flexibilidade para aumentar ou reduzir a capacidade rapidamente.

Exemplos de serviços em nuvem

Armazenamento de arquivos Google Drive, OneDrive, Dropbox
Hospedagem e servidores virtuais Amazon EC2, Azure Virtual Machines
Bancos de dados gerenciados Amazon RDS, Cloud SQL
Computação serverless AWS Lambda, Azure Functions
Plataformas de hospedagem Vercel, Netlify, Heroku
